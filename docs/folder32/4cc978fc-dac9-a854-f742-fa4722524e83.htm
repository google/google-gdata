Maximum
the Maximum specifies how many entries should be retrieved in total. This works together with . If set, AutoPaging of a feed will stop when the specified amount of entries was iterated over. If Maximum is smaller than PageSize (), an exception is thrown. The default is -1 (ignored).
OAuthDomain

OAuthUser

PageSize
the pagesize specifies how many entries should be retrieved per call. If not set, the server default will be used. Set it either to -1 (for default) or any value > 0 to set the pagesize to something the server should honor. Note, that this set's the max-results parameter on the query, and the server is free to ignore that and give you less entries than you have requested.
PrivateKey

Timeout
gets and sets the Timeout property used for the created HTTPRequestObject in milliseconds. if you set it to -1 it will stick with the default of the HTTPRequestObject. From MSDN: The number of milliseconds to wait before the request times out. The default is 100,000 milliseconds (100 seconds).
